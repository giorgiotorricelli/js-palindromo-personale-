# esercitazio PERSONALE sui palindromi come visto in classe

per verificare se una parola era un palindromo in classe abbiamo
creato un ciclo for che partiva dall'ultima lettera della parola
fino alla prima lettera, sommando ogni lettera ad una nuova stringa
vuota e infine verificando che la nuova stringa fosse identica alla
parola iniziale.
Abbiamo discusso con Samuel di provare a ricreare il programma però 
SEZIONANDO la parola a metà e controllando che le due metà invertite 
siano uguali.
